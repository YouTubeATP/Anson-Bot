# Anson-Bot

Anson-Bot is a bot for WhatsApp. It's currently in Alpha.<br>Its prefix is `a!`.

## To run
`tsc index.ts`<br>`node index.js`

## Features
- Dick command
- Eightball command
- Cat command
- Commands for MR (my own Minecraft railway)
- Choosing a random item
- Remind me (WIP)
- osu! statistics
